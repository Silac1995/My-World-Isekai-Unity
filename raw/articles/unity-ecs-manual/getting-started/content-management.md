# Content Management | Entities 6.4.0

## Overview

Unity's entity component system (ECS) includes "its own content management and delivery system" that provides efficient loading and releasing of Unity objects and scenes in data-oriented applications. The API works with both ECS systems and MonoBehaviour code, including bakers.

> "This system is built on top of Unity's ContentLoadModule assembly."

The content management documentation covers:
