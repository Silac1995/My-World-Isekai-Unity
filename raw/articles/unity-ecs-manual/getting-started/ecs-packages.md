# ECS Packages - Unity Entities 6.4.0

## Overview

Unity's data-oriented technology stack (DOTS) encompasses several packages and engine components that work together to enable high-performance development. The primary components include:

### Core Components

- **Entities** (this package): An implementation of the Entity, Component, System (ECS) architectural pattern.
- **Job system**: A multithreading solution designed for fast, safe concurrent code execution.
- **Burst compiler**: A C# compiler that produces highly optimized machine code.
- **Collections**: A collection of unmanaged data structures including lists and hash maps, essential for jobs and Burst-compiled contexts.
- **Mathematics**: A specialized math library with performance optimizations in Burst compilation.

### Built-in ECS Packages

- **Unity Physics**: A deterministic, stateless physics system for entities.
- **Havok Physics for Unity**: A deterministic, stateful physics system for entities.
- **Netcode for Entities**: Client-server networking infrastructure for entity-based systems.
- **Entities Graphics**: Rendering using the scriptable render pipeline (SRP) for entities.
- **Character Controller**: Tools for building custom ECS-based character control systems.
