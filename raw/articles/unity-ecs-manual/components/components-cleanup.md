# Cleanup Components

## Overview

Cleanup components function similarly to standard components, but trigger distinctive destruction behavior. When an entity containing a cleanup component is destroyed, Unity automatically removes all non-cleanup components from that entity. The entity persists until you eliminate all cleanup components from it. These components serve as tags to mark entities requiring cleanup operations upon destruction.
