# Enableable Components | Entities 6.4.0

## Overview

Enableable components allow you to disable or enable individual components on an entity during runtime. According to the documentation, this approach is particularly valuable when dealing with "states that you expect to change frequently and unpredictably because they create less structural changes than adding or removing components."

## Documentation Topics

The guide covers two main areas:

1. **Enableable Components Overview** - A foundational explanation of enableable components and their capabilities for runtime toggling
2. **Using Enableable Components** - Practical guidance on implementing and working with enableable components

## Key Concept

Rather than constantly adding and removing components to manage state changes, enableable components provide a lightweight mechanism to toggle component functionality on and off within a single entity instance.
