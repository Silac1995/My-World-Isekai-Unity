# Component Types | Entities 6.4.0

## Overview

The ECS system provides multiple component types to accommodate various use cases. This documentation section covers component types, their applications, and performance implications.

## Editor Icons

Unity represents different component types with distinct icons in relevant windows and inspectors:

- **Managed component**: Puzzle piece with curled brackets
- **Shared component**: Puzzle piece with share symbol
- **Tag component**: Puzzle piece with tag
- **Buffer component**: Puzzle piece with square brackets
- **Chunk component**: Puzzle piece with three squares

*Note: "Unity uses a generic component icon for component types not in the list."*

Generic component types display standard icons.
