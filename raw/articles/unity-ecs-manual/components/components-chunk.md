# Chunk Components

## Overview

A chunk component represents a specialized component type that stores values at the chunk level rather than per entity. According to the documentation, these components offer "similar functionality to shared components, but differ in some fundamental ways."

## Key Topics

The documentation covers three main areas:

1. **Introducing Chunk Components** — Learn the fundamentals and practical applications of chunk components

2. **Creating Chunk Components** — Instructions for implementing a new chunk component within your project

3. **Using Chunk Components** — Details on chunk component-specific APIs and operational methods
