# Entities Unity Editor Reference

This documentation page serves as an overview of the Entities-specific tools available within the Unity Editor for version 6.4.0.

## Overview

According to the page, "You can view information about the ECS elements in your project in Entities-specific Unity Editor windows." The documentation covers the various windows, inspectors, and workflows that support ECS development.

| Topic | Purpose |
|-------|---------|
| Authoring and runtime data mode | Explains the different data modes available in the Editor |
| Entities Preferences reference | Details Entities-specific properties in the Preferences window |
| Entities Project Settings reference | Outlines Entities-specific properties in the Player Settings window |
| Entities windows | Reference documentation for Editor windows, including the Entities Hierarchy window |
| Entities Inspectors | Information about Entities-specific Inspector tools |
