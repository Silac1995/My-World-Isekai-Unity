# Component Inspector Reference

## Overview

When you select a component, the Inspector displays its information across two distinct tabs:

- **Attributes:** Shows details about the component's C# type, including its namespace, member fields, and allocated chunk size.
- **Relationships:** Lists all Entities containing the selected component and all systems that query the component, organized per world.

![Component Inspector display showing Attributes on the left and Relationships on the right](images/editor-component-inspectors.png)

## Navigation

Each system or entity name includes an icon that allows you to navigate directly to that item. Clicking this icon changes the selection and opens the corresponding System Inspector or Entity Inspector when available.
