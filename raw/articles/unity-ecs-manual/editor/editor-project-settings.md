# Entities Project Settings Reference

## Overview

The Entities Project Settings allow you to configure Entities-specific options for your project. Access these settings by navigating to **Edit** > **Project Settings** > **Entities**.

## Available Settings

| Setting | Description |
|---------|-------------|
| **Excluded Baking System Assemblies** | Add assembly definition assets to prevent particular bakers from executing during your project build process. |
