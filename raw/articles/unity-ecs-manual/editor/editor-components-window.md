# Components Window Reference

## Overview

The Components window displays a list of C# component types that Unity has loaded across all assemblies and packages. This tool helps developers locate and investigate components throughout their project.

**Access:** Window > Entities > Components

## Functionality

The window provides the following capabilities:

- **Search functionality:** Use the search bar to find specific components
- **Component selection:** Select a component to view detailed information in the Component Inspector
- **Investigation tool:** Useful for exploring and understanding components in your project

For additional details on component inspection, refer to the Component Inspector reference documentation.

## Component Type Icons

The following icons represent different component types in the window:

| Icon | Component Type |
|------|----------------|
| Puzzle piece with curled brackets | Buffer component |
| Puzzle piece with share symbol | Chunk component |
| Puzzle piece with tag | Managed component |
| Puzzle piece with square brackets | Shared component |
| Puzzle piece with three squares | Tag component |
