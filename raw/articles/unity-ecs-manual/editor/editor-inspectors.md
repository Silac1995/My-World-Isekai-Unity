# Entities Inspectors

The Inspector feature displays comprehensive information about selected entities, systems, or components within the Unity Editor. The Entities-specific Inspectors provide specialized tools for working with ECS architecture.

## Available Inspector References

| Inspector Type | Purpose |
|---|---|
| Component Inspector | "displayed when you select a component" |
| Entity Inspector | "displayed when you select an entity" |
| System Inspector | "displayed when you select a system" |
| Query Window | Accessible via the Relationship tab of the Entity Inspector |

The documentation provides detailed reference materials for each inspector type, enabling developers to understand and interact with entity component systems effectively within the editor environment.
