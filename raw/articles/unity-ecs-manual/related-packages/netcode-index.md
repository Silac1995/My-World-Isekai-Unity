# Unity Netcode for Entities Manual (v6.6.0)

## Overview

"The Netcode for Entities, part of the Unity's Data Oriented Technology Stack (DOTS), provides a server authoritive with client prediction framework" for multiplayer game development.

The documentation references publicly available sample projects to help developers get started with introductory and advanced implementations.

## Installation & Compatibility

**Supported Unity Versions:**
- 2022.3 (LTS)
- Unity 6 (LTS)

To use this package, developers need a supported Unity version and foundational knowledge of "Unity's Entity Component System (ECS)" since the package builds upon it.
