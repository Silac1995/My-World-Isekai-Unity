# Netcode for Entities Project Setup

## Unity Editor Version

"Netcode for Entities requires you to have Unity version **2022.3.0f1** or higher."

## IDE Support

The Entities package utilizes Roslyn Source Generators. For optimal development experience, use an IDE with source generator compatibility:

- Visual Studio 2022+
- Rider 2021.3.3+

## Project Setup

Follow these steps to configure your project:

1. Launch **Unity Hub** and create a new **URP Project**
2. Open the **Package Manager** via **Window** > **Package Manager**
3. Use **Add package by name** (+ menu, top left) to install:
   - `com.unity.netcode`
   - `com.unity.entities.graphics`

After completing these steps, proceed with the next steps in the documentation.
