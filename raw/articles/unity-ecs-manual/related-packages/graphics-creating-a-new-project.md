# Install Entities Graphics - Creating a New Project

## Overview

Entities Graphics gathers rendering data for entities and integrates with Unity's rendering systems. The package works with the Universal Render Pipeline (URP) and High Definition Render Pipeline (HDRP), but is incompatible with the Built-in Render Pipeline.

## Create a New Project with Entities Graphics

The easiest installation method uses Unity's built-in project templates, which pre-configure necessary packages and settings.

### Steps

1. **Open Unity Hub** and start a new project
   - For URP workflows, select the **3D (URP)** template
   - For HDRP workflows, select the **3D (HDRP)** template

2. **Install Entities Graphics package**
   - Navigate to **Window > Package Manager**
   - Add the package by name: `com.unity.entities.graphics`
   - This automatically installs all dependencies, including ECS-related packages
