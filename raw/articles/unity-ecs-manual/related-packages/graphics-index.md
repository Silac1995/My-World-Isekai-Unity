# Entities Graphics Manual Index

## Overview

Entities Graphics is a rendering system for ECS entities that integrates with Unity's existing rendering infrastructure. The package "provides systems and components for rendering ECS entities" and works alongside the Universal Render Pipeline (URP) and High Definition Render Pipeline (HDRP) for content authoring and rendering passes.

## Main Sections

The manual covers:

- **Requirements and Compatibility** – System requirements and version support information
- **Getting Started** – Introductory guide for implementing Entities Graphics
