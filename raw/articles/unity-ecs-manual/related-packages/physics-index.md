# Unity Physics Overview

## Content Summary

The Unity Physics package is described as "part of Unity's Data-Oriented Technology Stack (DOTS), provides a deterministic rigid body dynamics system and spatial query system."

The documentation indicates support for two long-term support versions: 2022.3 (LTS) and Unity 6 (LTS).

Installation requires accessing the Package Manager through the Unity editor menu and adding the package either by its identifier (`com.unity.physics`) or via Git URL.

The page references the [Unity Physics Samples](https://github.com/Unity-Technologies/EntityComponentSystemSamples/tree/master/PhysicsSamples) repository for tutorials and example projects.
