# Defer Data Changes

This documentation page provides guidance on using entity command buffers to manage deferred data modifications in Unity's Entities system.

## Overview

The page explains that developers can "use entity command buffers to defer data changes, such as structural changes." This approach allows for managing modifications that need to happen outside the normal execution flow.

## Key Topics Covered

The documentation is organized around four main areas:

1. **Entity Command Buffer Overview** — Introduces the concept of deferring data changes with entity command buffers

2. **Using Entity Command Buffers** — Provides practical guidance on implementing entity command buffers in code

3. **Entity Command Buffer Playback** — Explains how to execute buffered commands within job systems

4. **Automatic Playback and Disposal** — Describes automated mechanisms for processing and cleaning up entity command buffers
