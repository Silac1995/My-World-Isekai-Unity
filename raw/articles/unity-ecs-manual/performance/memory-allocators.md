# Memory Allocators

The Entities documentation describes various memory allocation approaches for Unity applications. The page explains that developers can "use different memory allocators to manage memory in your application. You can use allocators in the Collections namespace, plus specific Entities built-in allocators."

## Key Topics Covered

The documentation organizes memory allocation guidance into several focused areas:

- **Memory allocators overview**: Foundational concepts for managing and allocating memory
- **Prebuilt custom allocators overview**: Using Entities-specific allocators for memory assignment
- **World update allocators**: Memory management strategies across different worlds
- **Entity command buffer allocators**: Memory handling for entity command buffers
- **System group allocators**: Memory management for different system groups
