# ECS Workflow Examples

## Overview

This documentation page presents Unity's Entity Component System (ECS) workflows, focusing on a "data-oriented approach to coding in Unity."

## Main Topics

The page outlines six key workflow areas:

| Topic | Purpose |
|-------|---------|
| Introduction to the ECS workflow | Foundational concepts for ECS application development |
| Starter ECS workflow | Basic patterns to begin working with ECS |
| Authoring and baking workflow | Creating and preparing content for ECS systems |
| Prefab instantiation workflow | Methods for duplicating prefabs within ECS |
| Make a system multithreaded | Techniques for parallel job scheduling across multiple threads |
| Use entity command buffer for structural changes | Strategies for ECB usage during entity iteration |

## Additional Resources

The documentation references an ECS sample project available on GitHub for practical reference implementations.
