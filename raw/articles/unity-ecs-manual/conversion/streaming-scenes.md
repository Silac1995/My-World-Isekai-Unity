# Scene Streaming Documentation

This page serves as a hub for Unity Entities scene streaming documentation (version 6.4.0). The content explains that "all scene loading in Entities is asynchronous. This is called **streaming**."

## Overview

Scene streaming enables developers to manage scene loading without performance interruptions. The documentation indicates you can "control how scenes are streamed with sections, meta entities, and instancing."

## Main Topics Covered

The page organizes information into five primary sections:

1. **Scene streaming overview** — foundational concepts for understanding the streaming system
2. **Scene section overview** — techniques for working with scene sections
3. **Load a scene** — implementation steps for loading scenes in your project
4. **Scene and section meta entities** — understanding meta entity systems
5. **Scene instancing** — creating multiple scene instances
