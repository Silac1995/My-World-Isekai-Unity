Speech Bubble Animation Style

How should bubbles appear and disappear?

A

Scale Pop

Bubble scales from 0 → 1 (with slight overshoot bounce). On exit, scales back to 0. Classic cartoon feel.

Hello
↔ bouncy entrance
B

Fade + Slide Up

Bubble fades in while sliding slightly upward from below. On exit, fades out while sliding up. Smooth and subtle.

Hello
↑ gentle float in
C

Scale Pop In + Fade Out

Entrance uses bouncy scale pop for impact. Exit uses a gentle fade-out for subtlety. Best of both worlds.

Pop!
→ then fades out gently
